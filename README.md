# Learn Playwright 2X

This repository contains learning materials and code examples for JavaScript fundamentals and Playwright automation testing.

## 📁 Repository Structure

```
LearnPlaywright2X/
└── chapter_01_Basics/
    ├── Basics.js             - Basic JavaScript examples
    ├── Commennt.js           - Comment types and shortcuts
    ├── Identifier.js         - JavaScript identifier rules
    └── NamingConvention.js   - Naming conventions (camelCase, PascalCase, snake_case, Hungarian)
```

## 📚 Chapter 1: JavaScript Basics

This chapter covers the fundamental building blocks of JavaScript:

| File | Topic |
|------|-------|
| `Basics.js` | Basic syntax and `console.log` |
| `Commennt.js` | Single-line (`//`) and multi-line (`/* */`) comments |
| `Identifier.js` | Rules for valid variable names, case-sensitivity, Unicode |
| `NamingConvention.js` | camelCase, PascalCase, snake_case, and Hungarian notation |

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ksagar0072-commits/LEARNPLAYWRIGHT2X.git
   ```

2. Navigate to the project folder:
   ```bash
   cd LEARNPLAYWRIGHT2X
   ```

3. Run any JavaScript file with Node.js:
   ```bash
   node chapter_01_Basics/Basics.js
   ```

## 📝 Naming Conventions Covered

| Convention | Example | Use Case |
|------------|---------|----------|
| **camelCase** | `firstName` | Variables, functions |
| **PascalCase** | `FirstName` | Classes, constructors |
| **snake_case** | `first_lastname` | Constants, config keys |
| **Hungarian** | `strName` | Legacy / type prefixing |
